# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.1] - 2026-02-02

### Fixed
- Critical security patch ensuring dependencies are clean.
- Minor configuration adjustments for production environment.

## [1.0.0] - 2026-02-02

### Added
- Initial release of SkillTest platform
- Multi-role authentication system (Admin, Recruiter, Candidate)
- Test creation and management interface
- Real-time test timer with anti-cheat detection
- Automatic scoring engine
- PDF certificate generation
- Stripe payment integration
- Comprehensive dashboards for all user roles
- Manual review system for low-scoring tests
- Custom notification system
- Adaptive difficulty engine (Beta)
- Docker containerization
- CI/CD pipelines with GitHub Actions

### Features
- Angular 18 frontend with Signals API
- Node.js/Express backend with TypeScript
- SQLite database with Sequelize ORM
- JWT authentication with Bcrypt password hashing
- RGPD compliance
- Responsive design with Tailwind CSS
- Glassmorphism UI design
- Multi-tenant architecture ready

### Security
- Role-based access control
- Password hashing with salt
- JWT token validation
- Anti-cheat detection system
- Secure API endpoints

### Documentation
- Complete technical documentation
- API documentation
- User guides
- Testing procedures
- Deployment instructions
